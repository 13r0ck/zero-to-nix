The first step is to install [Harmonic]:

```shell
curl https://harmonic.dev | sh
```

<Topic id="harmonic" />

## Install Nix

With Harmonic installed, check to make sure that you can install Nix on your machine:

<Harmonic command="plan" />

With Harmonic installed, you can now install Nix:

<Harmonic command="install" />

## Verify

To verify that the Nix CLI has been installed:

```shell
nix --help
```

You should see help output that begins like this:

```shell
Name

    nix - a tool for reproducible and declarative configuration management
```
