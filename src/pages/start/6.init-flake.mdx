Supports a wide variety of [languages and tools][handlers]

The [`fh`][fh] CLI

Some examples:

* If your project has a [`Cargo.toml`][cargo-toml] file in the root, `fh init` infers that it's a [Rust] project and asks if you want to add Rust dependencies to your Nix [development environment][dev-env].
