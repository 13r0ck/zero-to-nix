[Nix Installer][install] is an unofficial, opinionated installer for Nix created by [Determinate Systems][ds]. It currently supports both Linux and macOS.

<Brief id="nix-installer-differences" />

## Using Nix Installer

You can run Nix Installer on your system by with this command:

```shell
curl --proto '=https' --tlsv1.2 -sSf -L https://install.determinate.systems/nix | sh -s -- install
```

For a more complete guide, see [Get Nix running on your system][start] here in Zero to Nix.
